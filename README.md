## Colores

Colores es una Pequeña aplicación web creada para tener un listado de colores usados en un proyecto y poder copiarlos de forma rápida.

### Tecnologías usadas

- ReactJS como framework para el front end.
- MaterialUI para botones e iconos.

### Cómo usar colores de forma local

Si quieres usar Colores de forma local, asegúrate de tener npm instalado. Si los tienes, solo debes clonar este repositorio en tu computadora. Una vez clonado, abre el tu terminal en el directorio en el que se encuentra el proyecto, y corre el comando:

```shell
npm install
```

Esto instalará las dependecias necesarias para ejecutar el proyecto. Luego de que estén instaladas, puedes iniciar la app en forma local haciendo:

```shell
npm start
```

ó

```shell
yarn start
```
